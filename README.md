# svgvgkbr
Сайт РОО «Союз ветеранов Группы войск в Германии» по Кабардино-Балкарской Республике.

## Локальный запуск
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
```

## Публикация на GitHub Pages
В репозитории настроен workflow `.github/workflows/deploy.yml`.

Он автоматически:
1. Устанавливает зависимости.
2. Собирает сайт (`npm run build`).
3. Публикует `dist/` на GitHub Pages.

Публикация запускается при push в ветки `main`, `master` и `work`, а также вручную через **Actions → Deploy to GitHub Pages**.

## Блог
- Страница блога доступна по адресу `#/blog`.
- Вход в блог:
  - Логин: `svgvgkbrartur`
  - Пароль: `5.Atmrsri`
- После авторизации в блоге можно создавать новые публикации прямо из интерфейса блога.
- Поддерживаются вложения: изображения, видео и другие файлы (документы и архивы).
- Публикации из блога автоматически отображаются на главной странице в разделе «Новостная лента».
- В новостной ленте на главной странице отображается текст публикации и предпросмотр вложений (фото/видео/файлы).
- Кнопка «Читать» на главной открывает полупрозрачное окно для удобного чтения полной новости и просмотра вложений слева.
- Фото/видео из модального окна открываются в отдельном режиме просмотра с ручным масштабированием для изображений.
- Данные публикаций и вложений сохраняются в `localStorage` браузера.
